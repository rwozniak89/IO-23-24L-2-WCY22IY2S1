# Projekt: 7 - Kino

## Opis projektu
Projekt obejmuje aplikację mobilną kina. Ma ona na celu umożliwienie użytkownikom łatwe przeglądanie repertuaru kinowego, rezerwowanie biletów oraz uzyskiwanie informacji o filmach i seansach.
